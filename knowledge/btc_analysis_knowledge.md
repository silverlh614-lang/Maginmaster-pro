## 1. 핵심 운영 원칙 (메타 레이어 — 가장 중요) [DURABLE]

이 세션 전체를 관통하는 인식론적 규칙. 새 분석을 할 때 이 렌즈를 먼저 적용한다.

1. **점 추정은 착시다.** BTC 가격의 단일 목표가는 본질적으로 노이즈에 선을 긋는 일. 항상 **분포·시나리오**로 답하고, 신뢰구간과 확률을 함께 제시한다.
2. **공적분(cointegration)이 정당성의 시금석이다.** 두 우상향 시계열의 높은 상관계수(R²)는 대부분 spurious. M2-BTC는 공적분을 통과해 정당하고(§5.2), S2F는 통과하지 못해 무의미하다(§4). 상관 이전에 공적분을 물어라.
3. **In-sample 적합 ≠ out-of-sample 예측력.** S2F는 R²≈0.95의 화려한 in-sample 적합에도 2021년 이후 완전히 붕괴했다. "얼마나 잘 맞았나"가 아니라 "표본 밖에서 견디나"를 본다.
4. **바닥을 점으로 찍지 말고 구간에 분할 진입한다.** 어떤 바닥 모델도 정확한 저점을 맞힌 적 없다. 사다리(ladder) 매집은 저점을 못 맞혀도 작동하도록 설계된다.
5. **모델은 가정의 함수다 (GIGO).** 렌즈 범위·가중치가 곧 주관적 인코딩. 결론을 신뢰하기 전에 가정의 민감도를 본다.
6. **감쇠(diminishing returns)가 사이클의 지배 법칙이다.** 모든 배수·ROI는 시총 성장에 따라 기하급수적으로 축소된다. 과거 배수를 그대로 외삽하면 반드시 과대추정한다.
7. **시장 성숙 = 드라이버 교체.** 4차 사이클부터 가격의 주 동력이 '반감기 공급충격'에서 '유동성·기관 자금'으로 이동. 반감기는 이제 타이밍 신호가 아니라 방향성 촉매.
8. **표본은 3개뿐이다.** 완결된 사이클은 2012·2016·2020 세 개. n=3으로 말하는 모든 "법칙"은 통계적으로 취약하다. 겸손을 기본값으로.

### 3.1 핵심 패턴 [DURABLE]
- **ROI 감쇠는 등비수열**: 96 → 30 → 7.9 → 2배, 매 사이클 **÷ 약 3.7배**. 로그 스케일에서 거의 직선.

- **사이클 타이밍**: 저점→고점 약 **1,060일**, 고점→저점 약 **370일**.
- **낙폭도 감쇠 중**: -85 → -84 → -77% → (이번 -51% 진행). 변동성 구조적 하락(60일 변동성 2012년 200%+ → 현재 ~50%).
- **4차는 선례를 깼다**: 역사상 처음 **반감기 이전에 신고가**($73k, 2024-03) — 2024-01 현물 ETF 승인 때문. 반감기 후 즉각 랠리 대신 5개월 횡보.

## 4. S2F 모델 — 왜 쓰지 않는가 (반면교사) [DURABLE]

PlanB의 Stock-to-Flow. **분석에서 배제하되, 실패 사례로 기억한다.**

- **정의**: S2F = 총재고 / 연간 신규발행. 희소성 기반.
- **명성**: 2010~2021 in-sample R²≈0.95, "계단"마다 가격이 모델선 추종.
- **왜 실패했나**:
  - Spurious regression: 두 우상향 비정상 시계열 회귀 → 인과 없이 R² 인플레(Granger-Newbold).
  - Auto-correlative: 예측선이 그것이 예측한다는 과거 가격들로 형성됨.
  - 공적분 실패: 후속 분석에서 두 시계열 비공적분 → 회귀 무의미.
  - 수요 완전 무시: 공급만 모델링.
- **붕괴 실측**: 2021말 예측 $100k vs 실제 $47k → 2022 $15.5k(예측比 -80%+) → 2025-26 예측 $500k vs 실제 $70k대. CoinGecko: 2025년 "largely defunct".
- **교훈**: §1-2, §1-3의 산 증거. 곱셈/희소성 곡선의 단순 외삽을 경계하라.

## 5. 2028 반감기 가격 프레임워크 (non-S2F) [혼합]

**2028 반감기**: ~2028-04 (블록 950,000), 보상 3.125→1.5625.
S2F 계열 제외, 성격이 다른 5개 렌즈로 조준.

### 5.1 파워로 (Power Law) — 가장 강한 정량 모델 [DURABLE 방법 / SNAPSHOT 값]
- **공식**: `P = A × (제네시스 이후 일수)^b`
  - Porkopolis: A=1.6×10⁻¹⁷, b=5.77
  - BGeometrics: A=1.0117×10⁻¹⁷, b=5.82
- **성격**: '시간 vs 가격' (S2F의 '희소성 vs 가격'과 완전히 다른 축). log-log에서 15년째 직선.
- **밴드**: 하단 = 0.42× 공정가치 (역사상 1회만 이탈, 2020-03 코로나), 상단 = 3× 공정가치.
- **2028 반감기(~7,047일) 계산값**: 공정가치 **~$250k~290k** / 하단 ~$105k / 상단 ~$750k.
- **강점**: 아직 한 번도 무효화 안 됨. "2028년 이후 $100k 밑으로 안 감" 주장.
- **약점**: fit 구간 민감(2년 vs 15년 계수 상이), 스케일 불변성은 가정이지 법칙 아님.

### 5.2 글로벌 M2 유동성 — 현재 실제 드라이버 [DURABLE 관계 / SNAPSHOT 값]
- **공적분 통과** (S2F와의 결정적 차이): 장기 탄력성 **2.65** (M2 1%↑ → BTC 2.65%↑), 오차수정항 ECT=-0.12 (괴리 월 12% 회복).
- VanEck: 상위 5개 통화 M2 변화가 BTC 분산 **54% 설명**(r²=0.54). 장기 기본 CAGR **15%**.
- [SNAPSHOT] M2 대비 Z-score: 2025-01 +1.48(고평가) → 2026-02 **-1.31(저평가)**.
- **2028 함의**: 순수 유동성 추세(15% CAGR)만으론 현재가 기준 **~$78k** (사이클 진폭 제외한 보수적 하단 앵커).

### 5.3 금 대비 상대가치 (TAM) — 수요측 사이징 [SNAPSHOT 값]
- 금 시총 ~$34조, BTC ~$1.2~1.5조 (금의 **약 4%**). 2028 유통량 ~19.9M.

| BTC가 금의 X% | BTC 가격 |
|---|---|
| 10% | ~$171k |
| 15% | ~$256k |
| 25% | ~$427k |
| 50% | ~$854k |

- 온건 가정(4%→8~15%): **~$135k~256k**. 단 금 자체가 ATH 행진 → 분모 변동 변수.

### 5.4 순수 사이클 타이밍 [DURABLE]
- 2025-10 고점 기준: 다음 저점 ~2026-10, 다음 고점 ~2029-09.
- → **2028-04 반감기는 회복 중반, 사이클 천장보다 한참 전**에 위치.

### 5.5 종합
| 렌즈 | 2028 반감기 함의 |
|---|---|
| 파워로 공정가치 | $250k~290k (하단 $105k / 상단 $750k) |
| M2 유동성 (CAGR) | $76k~80k (보수·추세만) |
| 금 대비 8~15% | $135k~256k |
| 사이클 타이밍 | 회복 중반, 천장 아래 |

- **구조적 수렴**: 파워로 공정가치 + 금 10~15% → **$170k~290k**.
- **핵심 변수**: 유동성 렌즈($78k)와 파워로($250k)의 3배 격차 = "사이클 진폭이 유동성 추세를 얼마나 초과하느냐". 불장 국면에 걸리면 파워로 쪽, 약한 사이클이면 유동성 쪽.
- **정직한 중심값**: ~$180k~220k, 신뢰구간 $80k(약한 사이클) ~ $500k+(강세 재현).

## 6. 바닥 예측 방법론 [혼합]

**목표: 바닥 시점·가격을 확률로 추정.** 온체인 지지선이 가장 실증적인 도구.

### 6.1 6개 렌즈 삼각측량 [방법 DURABLE / 값 SNAPSHOT]

| 방법 | 논리 | 바닥 추정 |
|---|---|---|
| ① 실현가격 지지 | 과거 바닥은 실현가격 근처/약간 아래 | $45k~54k |
| ② 200주선 | 사이클 바닥의 준-완벽 지지 | $55k~62k |
| ③ 낙폭 감쇠 | 완만한 -55~60% | $50k~57k |
| ④ MVRV 캐피튤레이션 | 진짜 바닥은 실현가격 하회 | $40k~53k |
| ⑤ 데스크로스 마지막 하락 | 3D 50/200 크로스 후 -46~52%(2014·17·22) | $29k~33k |
| ⑥ 과거 낙폭 재현 | -77~85% 그대로 | $25k~31k |

- ①~④(온체인·구조) **$45~57k** 촘촘히 수렴 / ⑤~⑥(모멘텀·역사반복) **$25~33k**. → 이봉(bimodal).

### 6.2 결정적 단 하나의 선: **실현가격 $53,600** [SNAPSHOT]
- **사수 시** → 바닥 $50~57k, 성숙 시장의 완만한 약세로 종결.
- **확신 이탈 시** → 첫 진짜 캐피튤레이션, $40k 붕괴 후 $33~45k, 최악 $29~33k.
- 이 값을 매일 모니터링. regime 스위치의 트리거.

### 6.3 타이밍 [DURABLE 방법]
- 고점→저점 ~370일 → 2025-10 고점 기준 **바닥 후보 ~2026-10(4분기)**. 지금(7월)은 바닥 국면 초입일 가능성.

### 6.4 바닥 확인 신호 (점 추정보다 중요) [DURABLE]
동시 발생 시 바닥 확률 급등:
1. MVRV Z-Score 녹색(과거 바닥) 진입 후 반등
2. 가격이 실현가격 회복 + 그 위 주간 마감
3. **LTH-MVRV > STH-MVRV 상향 크로스** (역사적 바닥/불장 시작 신호)
4. 200주선 재탈환
5. 극단적 공포인데 신저점 미형성 다이버전스
6. (보조) **Thermocap Z-Score 저평가 밴드(≤ -0.3) 진입** — 시총÷채굴자 누적보상,
   실현가격·MVRV와 같은 원가 계열의 사촌 지표. ⚠️ 감쇠 주의(§1-6): 천장 Z가
   +3 → +2 → +1.3으로 축소돼 왔으므로 **이번 사이클은 -0.3 미도달 반등 가능** —
   단독 트리거 금지, 동시신호의 한 표로만. (2026-07 현재 ~+0.2 중립, 하락 중)

## 7. 몬테카를로 앙상블 + 진입 FSM (구현) [DURABLE]

구현 파일: `btc_bottom_model.py` (별도 제공). 아래는 재사용 스펙.

### 7.1 Part 1 — 앙상블 바닥 분포 (mixture-of-experts)
각 렌즈 = 삼각분포(min, mode, max) + 신뢰도 가중치. 추첨마다 가중치∝ 렌즈 선택 → 삼각분포 샘플.

```
LENSES (min, mode, max, weight):
  realized_price     : (44000, 51000, 54000, 0.22)
  ma_200w            : (55000, 59000, 62000, 0.20)
  drawdown_decay     : (50000, 53500, 57000, 0.15)
  mvrv_capitulation  : (40000, 48000, 53000, 0.18)
  death_cross_leg    : (29000, 31000, 33000, 0.15)
  historical_repeat  : (25000, 28000, 31000, 0.10)
```

**실행 결과 (300k sims)** [SNAPSHOT — 위 값 기준]:
- EV(mean) $46,586 / **median $49,653** / mode $30,967
- P5 $28,004 · P25 $32,916 · P75 $54,108 · P95 $59,704
- **P(바닥<$53,600)=72.9%**, P(<$50k)=51.9%, P(<$45k)=29.6%, P(<$40k)=P(<$35k)=25.0%
- **이봉형**: 얕은 군집 $50~55k(~75%) / 깊은 군집 $28~33k(~25%), 사이 $33~40k 갭.
- 주의: mode($31k)는 깊은 렌즈가 좁아 생긴 아티팩트. **robust 중심값은 median/EV**.

### 7.2 Part 2 — 바닥 감지 FSM (일별 신호 생성기)
입력: `price, realized_price, mvrv_z, ma_200w, lth_mvrv, sth_mvrv, made_new_low`

```
States: WATCH → CAPITULATION_WATCH → ACCUMULATE → CONFIRM → TREND
Transitions:
  WATCH → CAP_WATCH        : price < realized_price OR mvrv_z < 0
  CAP_WATCH → ACCUMULATE   : mvrv_z < -0.2 AND under cost basis
  ACCUMULATE (deploy)      : 신저점 OR mvrv_z<-0.3 마다 1/N 트랜치 투입
  ACCUMULATE → CONFIRM     : reclaim(price>realized & mvrv_z>0) AND LTH-MVRV>STH-MVRV
  CONFIRM (deploy 잔량)     : 남은 트랜치 전량 투입
  CONFIRM → TREND          : price > 200주선 → 추세 로직 인계
```
- 설계 철학: 저점을 **못 맞혀도** 구간 분할 진입으로 작동(§1-4).
- 순수 로직 ~40줄, numpy 불필요 → TS/Node 포팅 용이.

### 7.3 시스템 이식 가이드
